Dit zijn mijn Projecten

1. Java: rekenmachiene

Ik heb een rekenmachine in Java ontwikkeld met Processing om te begrijpen hoe calculaties online functioneren. Dit project heeft mij geleerd hoe ik functies creëer, functies implementeer en voor het eerst met voorlopige variabelen werk, waardoor mijn programmeervaardigheden versterkt zijn.

2. Html Css

In dit project heb ik een stijlvolle webshop gecreëerd, waarbij de kledingstlijl ligt op esthetische kleding. De website opent met een visueel aantrekkelijke showcase van modellen in de laatste trends, gevolgd door een georganiseerde catalogus van producten. Door het ontwerpen van deze webshop heb ik diepgaand gewerkt met CSS Flexbox en Grid, waardoor ik effectieve en responsieve lay-outs kon bouwen.

3. Javascript

Etch-a-Sketch

Yahtzee

In deze projecten heb ik mijn JavaScript-vaardigheden verfijnd door geavanceerde functies en arrays te gebruiken. Voor Etch-A-Sketch gebruikde ik event listeners en getElementById om een gebruikersinteractie mogelijk te maken. Met Yahtzee verdiepte ik mij in DOM-manipulatie, wat een heel erge uitdaging was. Mijn doorzettingsvermogen en onderzoek hebben geresulteerd in een werkend spel dat zowel de logica als de interface.

4. Php

Login

In mijn PHP-project heb ik de backend-ontwikkeling geleerd door een login systeem te ontwerpen met CRUD-functionaliteiten. Dit gaf me inzicht in hoe data verwerkt, opgevraagd, bijgewerkt en verwijderd kan worden in een database. Ik leerde hoe je veilig gegevens creëert, leest, aanpast en wist, wat essentieel is voor betrouwbare webapplicaties.
